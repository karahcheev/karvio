# Dataset Permissions and API

This page summarizes role requirements and API links for datasets.

## Permissions

| Action | Minimum Role |
| --- | --- |
| View datasets | `viewer` |
| Create and edit datasets | `tester` |
| Link or unlink a dataset from a test case | `tester` |
| Delete a dataset or bulk delete datasets | `lead` |

## API

Endpoint tables, query parameters, payload examples, revision behavior, binding modes, row selection, and bulk delete are documented in [Datasets API](../api/datasets/index.md).
